# Interview Tracker WebApp

A website where user is required to authenticate himself by registering on the server, after that he can pracitce questions related to CP and can also add some questions. He can also read the interview experiences of students and can add his own experience also.
<br>
This webapp also has a admin interface panel which is implemented using <b>AdminBro</b>. The main purpose of admin is to review existing models and approve the questions and experiences added by the user.
<br>
